Define Parameters and Variables¶

  • min_pitch - minimum pitch sung in the song (used for pitch tracking); corresponds to the lower Pa of the singer<\li>
  • max_pitch - max pitch sung in the song (used for pitch tracking); corresponds to the higher Pa of the singer<\li>
  • start_time - start time of the segment that is to be plotted (seconds)<\li>
  • end_time - end time of the segment that is to be plotted (seconds)<\li>
  • audio_path - path to the audio file to be analysed<\li>
  • cycle_file - path to the file with annotation of sums in the song<\li> <\ul>

audio, sr = librosa.load(audio_path, sr=None, mono=True, offset=start_time, duration = end_time - start_time)
snd = parselmouth.Sound(audio, sr)
pitch = snd.to_pitch_ac(0.01, min_pitch, 15, True, 0.03, 0.45, 0.01, 0.9, 0.14, max_pitch)
cycle_df = pd.read_csv(cycle_file)
sams = cycle_df.loc[(cycle_df['Time'] >= start_time) & (cycle_df['Time'] <= end_time)]

Plotting the pitch contour. Sams are marked with a vertical line and the cycle number is noted in a box at the bottom of the line.

xvals = pitch.xs() + start_time

yvals[yvals==0] = np.nan
fig, ax = plt.subplots(figsize=(14, 7))
ax = sns.lineplot(x=xvals, y=yvals, ax=ax)
ax.set(xlabel='Time Stamp (s)', ylabel='Pitch (Hz)', title='Pitch Contour', ylim=(min_pitch-20, max_pitch+20))
for i, sam in sams.iterrows():
    ax.axvline(sam['Time'], linestyle='--', c='purple')
    ax.annotate(sam['Cycle'], (sam['Time']-1, min_pitch-10), bbox=dict(facecolor='grey', edgecolor='white'), c='white')
